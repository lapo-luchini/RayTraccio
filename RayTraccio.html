This is my first attemp at Java programming, a SLOW (it has no great optimization, insofar) raytracing program that supports

  • One 3D shape
  • One Light
  • One Camera
  • First reflection
  • A number of shapes:
    • Quadric (generic support for spheres, paraboloids, hyperboloids, cones, ...)
    • Plane
    • CSG Union (draws various sub-shapes altogether)
    • CSG Intersection (intersecates the sub-shapes)
  • A number of procedural textures:
    • Plain
    • Checker (draws the two sub-textures checkered along all the axes)
    • Stripes (draws the two sub-textures divided in stripes along Y axes)
    • [yes, I know the last two are bugged as the stripe around zero is double-size... I'll correct it someday!]
